# Conservative Evangelical Systematic Theology Research Packet

## Purpose

This packet prepares Logos to support a conservative evangelical source spine that churches can recognize and trust.

It is staged only. It does not import copyrighted sources, create doctrine nodes, create claim objects, or ingest Bible text.

## Packet files

- `grudem-source-profile.md`
- `conservative-theologian-anchor-map.md`
- `bible-ingestion-raggraph-plan.md`
- `licensing-boundary-notes.md`
- `copyright-safe-source-options.md`

## Main thesis

The repo should support conservative evangelical retrieval and rendering by building:

> source profiles
> -> authority tiers
> -> doctrine order maps
> -> Bible source registry
> -> licensing policy
> -> open-text pilots
> -> claim/evidence objects
> -> RAGGraph routes


This avoids the failure mode of dumping copyrighted theology and Bible translations into a public AI repository.

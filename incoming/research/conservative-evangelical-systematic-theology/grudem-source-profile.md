# Wayne Grudem Source Profile

## Source status

Wayne Grudem's *Systematic Theology* is a high-value conservative evangelical source anchor.

It should be handled as:

> reference profile
> -> doctrine-order map
> -> chapter/page citation surface
> -> original summary in Logos prose
> -> no full-text public ingestion

## Why Grudem matters

Grudem is useful for Logos because he is recognized in many conservative evangelical churches; accessible to pastors and lay leaders; systematic rather than only topical; and clear on biblical authority, doctrine order, creation, humanity, sin, grace, church, and eschatology.

## Trust and scope notes

Grudem should not be treated as the only conservative evangelical source. Reception varies by denomination and doctrinal issue, especially around continuationism, complementarian theology, ecclesiology, baptism, eschatology, prophecy, charismatic gifts, and theological method.

For Logos, Grudem should be one profile within a broader conservative evangelical source spine.

## Safe public-repo treatment

Allowed:

- source profile;
- bibliography;
- doctrine-order map;
- chapter-level references;
- original summaries;
- short quotations only where appropriate;
- comparison against public-domain and open-license sources.

Not allowed without permission:

- full book text;
- chapter copies;
- bulk chunks;
- public embeddings built from the full text;
- a public RAG index containing the full copyrighted text.

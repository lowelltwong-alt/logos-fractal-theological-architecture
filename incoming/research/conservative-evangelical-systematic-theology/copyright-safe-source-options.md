## Practical source strategy

1. Build source profiles for modern trusted theologians.
2. Ingest public-domain theology for actual chunking.
3. Ingest public-domain/open Bible translations for actual chunking.
4. Keep modern copyrighted sources in reference-only or private licensed connectors.
5. Build original-language pipeline only after license review and qualified reviewer lanes.
6. Use open/public-domain evidence for public claim objects wherever possible.

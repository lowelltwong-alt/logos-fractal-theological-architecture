# Incoming Research Staging

## Purpose

This folder is a staging area for research packets that are not yet promoted into the governed doctrine, concept, application, or graph layers.

Research packets here may contain:

- source spines
- proposed claim inventories
- dependency maps
- candidate graph object plans
- Codex-ready implementation handoffs
- review questions for theological or governance stewards

## Status rule

Nothing in this folder should be treated as canonical doctrine, approved ontology structure, or production graph data.

The intended flow is:

```text
incoming research packet
-> human review
-> claim cleanup
-> evidence strengthening
-> proposed node or relationship objects
-> validation
-> promotion into docs/ or data/graph only after review
```

## Promotion rule

Research may be promoted only when the proposed object has:

- a stable node or claim identity
- explicit source basis
- clear asserted vs inferred status
- provenance note
- downstream dependency notes
- review status
- no uncontrolled vocabulary drift

# Human Person and Formation Core Slice

## Purpose

This packet stages the research and implementation plan for the next Logos feature branch:

```text

```

The slice deepens the repository's theological anthropology and formation base so future AI governance, institutional ethics, labor/vocation, and accountability work can derive from a stable human-person framework.

## Thinker anchors

Primary existing/first-ring anchor:

- Augustine

Core source anchors for this slice:

- Augustine: ordered love, fallenness, grace, interiority, institutions.
- Athanasius: incarnation, image renewal, restoration of human nature.
- Dallas Willard: formation, heart/will, intentional apprenticeship, human agency.
- Henri Nouwen: servant leadership, vulnerability, non-dominating power.
- Tim Keller: vocation/work, public faithfulness, work as service.
- Calvin/Luther: later anchors for grace, vocation, fallenness, work, and calling; do not overbuild them in this slice.

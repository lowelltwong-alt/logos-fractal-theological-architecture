# Claim Objects

## Purpose

This folder is the entry point for governed claim objects in the Logos repository.

Claims are first-order reasoning objects that preserve:
- subject
- predicate
- object
- epistemic status
- trust zone
- review posture
- evidence references
- lineage

## Why this layer exists

Important reasoning should not remain only in prose or in bare graph edges.

## Recommended first claims

The first reference-slice claims should connect:
- Genesis 1:26-28
- image of God
- anthropology
- stewardship
- vocation
- Hebrew lexical terms such as `tselem` and `demut`

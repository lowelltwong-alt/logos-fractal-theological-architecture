# Mark 1 primary-sources sample

## Purpose

This folder is a starter example of how the future primary-sources branch could work for a New Testament passage.

It uses Mark 1 as a compact Greek test case for:
- codex witness handling
- stable scholarly page/viewer linking
- transcription and passage reconstruction
- lexical evidence
- translation comparison
- graph linkage into theological themes

## Scope note

This sample is illustrative rather than exhaustive.

It is meant to complement the Genesis 11 sample so a future lead can see both:
- an Old Testament Hebrew pattern
- a New Testament Greek pattern

## Summary principle

A second example set helps demonstrate that the future primary-sources ontology is not tied to only one corpus, one language, or one witness style.

# Genesis 11 primary-sources sample

## Purpose

This folder is a starter example of how the future primary-sources branch could work for an Old Testament passage.

It uses Genesis 11:1–9 as a compact test case for:
- fuller witness handling
- fragmentary witness handling
- transcription
- passage reconstruction
- lexical meaning in context
- translation comparison
- graph linkage into a theological theme

## Notes on scope

This sample is illustrative rather than exhaustive.

It includes:
- a fuller Hebrew witness anchor
- a fragmentary Genesis witness example
- a transcription object
- a reconstruction object
- a lexical-evidence object
- an English translation-comparison object
- a governed relationship object

## Summary principle

This sample shows how a future corpus can move from witness or fragment to theology without collapsing source evidence, meaning, and translation into one layer.

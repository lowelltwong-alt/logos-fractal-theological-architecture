# Passage Nodes

## Purpose

This folder holds passage nodes for the concordance graph.

Passage nodes should be used when a larger textual unit carries the main semantic weight and should not be fragmented into verse-only analysis.

## Core rule

Use a passage node when:
- the relationship depends on a multi-verse unit
- a discourse, hymn, narrative arc, or argument is the real object of analysis
- a verse-level breakdown would lose the main meaning

## What belongs here

- stable passage identifiers
- higher-order semantic groupings of verses
- links to concepts, doctrines, motifs, and interpretation nodes when the larger unit is primary

## Summary principle

A passage node exists so the concordance can model larger scripture units without forcing everything into isolated verse references.

# Retrieval Neighborhoods

## Purpose

This folder is the entry point for governed retrieval neighborhoods and context packages in the Logos repository.

## Why this layer exists

The repository should eventually produce governed context packages rather than relying only on naive keyword retrieval or unconstrained graph adjacency.

## First recommended neighborhood types

- passage neighborhood
- doctrine neighborhood
- concept neighborhood
- comparison neighborhood
- application neighborhood

## First recommended reference implementation

Build a passage neighborhood for Genesis 1:26-28 that packages:
- the text node
- image of God theme node
- anthropology doctrine node
- stewardship and vocation concept nodes
- Hebrew lexical notes
- relevant translation and manuscript context
- claim objects connecting them

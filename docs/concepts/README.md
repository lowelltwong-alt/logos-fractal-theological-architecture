# Concept Layer

This folder is the entry point for reusable concept nodes that recur across doctrine, scripture, canon, and application work.

## Why this layer exists

Some concepts appear across multiple branches and should become independently retrievable rather than remaining buried inside one thinker page or one doctrine file.

## Recommended early concept buildout

The first high-value concept nodes for this repository include:
- imago Dei
- stewardship
- vocation
- institutions
- ordered love
- public reason

## Relation to the rest of the project

This layer connects:
- doctrine nodes
- biblical theme nodes
- canon thinker nodes
- scripture text nodes
- application and governance nodes

# Biblical Themes Layer

This folder is the entry point for recurring biblical themes that are not limited to one passage.

Examples include:
- image of God
- stewardship
- authority
- wisdom
- work
- fall
- redemption

## Why this layer exists

Biblical themes often connect multiple passages, books, doctrines, and thinkers.

This layer allows those recurring themes to become reusable nodes instead of remaining scattered across isolated text notes or theologian summaries.

## Relation to the rest of the project

This layer connects:
- scripture text nodes
- interpretation nodes
- doctrine nodes
- concept nodes
- canon thinker nodes

It is one of the main bridges between biblical studies and theological architecture.

## Recommended first themes

The strongest first buildout themes for this repository are:
- image of God
- stewardship
- fall
- vocation
- institutions
- wisdom

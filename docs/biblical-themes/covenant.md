# Covenant

## Why this theme matters

Covenant is one of the most important biblical themes in the repository because it organizes major questions of promise, obligation, peoplehood, mediation, continuity, discontinuity, authority, law, grace, and redemptive order.

For this project, covenant matters because different Christian traditions weight the covenants differently, and those differences shape theology, ecclesiology, public theology, governance, and institutional reasoning.

## Core scriptural clusters

This theme will eventually need to connect a wider scriptural network, including early major covenant passages such as:
- Genesis 12:1–3
- Genesis 15
- Genesis 17
- Exodus 19–24
- 2 Samuel 7
- Jeremiah 31:31–34
- Luke 22:20
- Hebrews 8

## Major interpretive questions

This theme raises several major questions:
- how the covenants relate to one another
- what continuity and discontinuity look like across redemptive history
- how law, promise, peoplehood, and mediation are ordered
- how covenant thinking shapes institutions, authority, and communal obligation
- how different traditions map covenant differently in doctrine and practice

## Downstream significance

This theme strongly informs:
- public theology
- ecclesiology
- political theology
- continuity and discontinuity debates
- institutions, authority, and obligation
- Christian reasoning about communal order and governance

## Repository use

In this repository, the covenant theme should function as a bridge between:
- biblical text clusters
- doctrine nodes
- canon thinker nodes
- tradition-specific interpretation
- governance and public-theology applications

## Recommended next expansion

This node will likely need later child nodes such as:
- Abrahamic covenant
- Mosaic covenant
- Davidic covenant
- New covenant

Those should be added only when the main covenant structure in the repository is ready to support them well.

# Noncanonical Sources Layer

This folder is the entry point for noncanonical, disputed, deuterocanonical, pseudepigraphal, forged, and heretical-source handling in the repository.

## Why this layer exists

The project needs a place to document important boundary texts without allowing them to blur into the canon, doctrine, or concept layers as though they were equivalent authorities.

This folder exists to preserve:
- historical awareness
- comparative theology
- reception history
- doctrinal boundary clarity

## Core rule

No source in this folder should be treated as doctrinal authority by default.

Every node here should be classified explicitly using the framework in:
- `docs/governance/noncanonical-and-heresy-classification.md`

## Recommended substructure

- `deuterocanonical/` — texts received in some Christian traditions but not others
- `pseudepigrapha/` — ancient noncanonical but historically relevant literature
- `forgeries-and-heretical/` — high-risk sources requiring explicit warning and strong use limits

## Interpretation rule

In this repository, noncanonical documentation exists to preserve careful boundaries, not to weaken them.

Some texts may be:
- historically valuable
- reception-history relevant
- useful for background or comparison

while still being:
- noncanonical
- doctrinally limited
- inappropriate for establishing theology

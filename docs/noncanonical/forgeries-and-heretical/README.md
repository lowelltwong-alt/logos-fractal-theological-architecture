# Forgeries and Heretical Sources

This folder is for high-risk texts that require the strongest boundary controls in the repository.

These sources may be documented for:
- historical awareness
- comparative theology
- boundary-setting
- reception history

They should not be used to establish doctrine.

## Required handling rule

Any node in this folder should include:
- explicit classification fields
- a warning block near the top of the file
- a clear statement of what the source may and may not be used for

## Default assumption

Unless explicitly overridden with strong justification, the default use-case limit for this folder is:
- `do_not_use_for_doctrine`

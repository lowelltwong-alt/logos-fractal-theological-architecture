# Pseudepigrapha and Related Literature

This folder is for ancient noncanonical literature of historical, thematic, or comparative importance.

Texts here may be useful for:
- historical context
- reception history
- thematic comparison
- understanding the background of some interpretive traditions

They should still be classified carefully and should not be treated as doctrinal authority by default.

# Sectarian and Disputed Translations

This folder is for translations that require stronger caution because of one or more of the following:
- sectarian control
- doctrinally tendentious renderings
- disputed textual-base claims
- unsupported or misleading provenance claims
- high manipulation risk

## Core rule

No translation in this folder should be treated as a primary doctrinal witness by default.

Each node should document why it is being classified here and what it may still be used for, such as:
- comparison
- movement history
- translation analysis
- doctrinal boundary-setting

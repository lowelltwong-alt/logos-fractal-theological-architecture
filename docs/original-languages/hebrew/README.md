# Hebrew Layer

This folder is the local entry point for Hebrew lexical and original-language work in the repository.

## Why this branch exists

The repository needs a place to preserve Hebrew lexical nuance where doctrine, biblical themes, scripture interpretation, translation, and governance implications depend on original-language distinctions.

This branch exists so those distinctions do not remain buried inside larger prose nodes.

## What belongs here

Examples of what belongs in this branch:
- lexical term nodes
- semantic-range notes
- translation-sensitive term notes
- Hebrew terms central to doctrine, anthropology, stewardship, covenant, wisdom, or authority

## Relation to the rest of the repository

This branch connects strongly to:
- scripture passages
- translation witness nodes
- manuscript witness nodes
- biblical-theme nodes
- doctrine and concept nodes

## Fractal rule for this branch

This branch should follow the same recursive pattern as the wider repository:
- local README
- governed term nodes
- stable anchors
- controlled tags
- provenance and review fields where relevant
- promotion of repeated patterns when needed

## Current starting terms

The early Hebrew lexical cluster already includes:
- `tselem`
- `demut`
- `radah`

Additional high-value terms may be added later as the scripture and doctrine layers deepen.

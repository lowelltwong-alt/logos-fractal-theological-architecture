# Trinity

## Purpose

This file defines Trinity as a doctrine topic for the Logos repository.

It is the stable theological object under consideration, not a final judgment about every Trinitarian formulation.

## Topic summary

The Trinity topic concerns how Christian theology speaks of the one God in relation to Father, Son, and Holy Spirit.

This topic should remain reusable across traditions, doctrinal formulations, and later assessment work.

## Pilot scope

This pilot topic is intentionally small.

It anchors:

- one doctrine topic
- three representative orthodox or near-orthodox doctrine views
- one clearly rejected doctrine view preserved as assessed history
- baseline-aware assessments attached to views rather than to the topic

## Starter scripture references

- Matthew 28:19
- John 1:1-18
- John 14:16-17
- 2 Corinthians 13:14

## Separation rule

This topic file does not declare which Trinity view is correct.

That work belongs in doctrine view files and doctrine assessment files.

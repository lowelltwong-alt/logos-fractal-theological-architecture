# Anthropology

## Purpose

This node defines the theological category of anthropology for the Logos repository.

## Why this node matters

Anthropology answers what a human being is, what dignity consists in, what creaturely limits apply, what responsibilities belong to persons, and what kinds of delegation or automation become dehumanizing.

Because this repository is concerned with theology, governance, institutions, and AI-assisted work, anthropology is one of the most important upstream doctrine nodes.

## Early grounding focus

The first buildout for this node should be grounded especially in:
- Genesis 1:26-28
- image of God theme work
- stewardship and vocation concepts
- lexical notes such as `tselem` and `demut`

## Downstream relevance

This node should eventually constrain:
- Christian AI anthropology work
- institutional decision design
- labor and automation reasoning
- dignity-sensitive governance logic

# Doctrine Layer

This folder is the entry point for stable doctrine-level objects in the Logos repository.

## Why this layer exists

Doctrine nodes preserve major theological objects as governed, reusable structures rather than leaving them scattered across thinker summaries, scripture notes, or applications.

## Recommended early doctrine buildout

The first high-value doctrine nodes for this repository include:
- anthropology
- creation
- sin
- grace
- vocation
- institutions

## Relation to the rest of the project

This layer connects directly to:
- scripture grounding
- biblical themes
- concept nodes
- canon thinker nodes
- downstream governance and application work

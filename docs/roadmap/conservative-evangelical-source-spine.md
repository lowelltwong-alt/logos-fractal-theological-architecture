# Conservative Evangelical Source Spine

## Purpose

This roadmap defines a conservative evangelical source spine for Logos.

The goal is not to make the repository sectarian. The goal is to define a scoped, reviewable profile that conservative evangelical churches, pastors, and contributors can recognize as serious and bounded.

This source spine should support later work on:

- systematic theology profile cards;
- doctrine/source routing;
- conservative evangelical rendering profiles;
- Bible source registry and licensing review;
- open-text Bible ingestion pilots;
- future claim objects and graph relationships.

## Boundary

Do not ingest modern copyrighted systematic theology books or modern Bible translations into this public repository unless there is a clear lawful basis.

Do not treat this roadmap as doctrine.

Do not promote summaries of modern copyrighted works into claim objects without source-basis review and citation-policy review.

## Why this is needed

Logos needs a source spine that can answer:

- Which theologians and biblical scholars should conservative churches recognize?
- Which sources may influence doctrine nodes?
- Which source texts can be chunked and indexed publicly?
- Which sources require private licensed connectors rather than public repository ingestion?
- Which Bible translations or editions are safe for public ingestion?
- Which original-language texts and editions are safe to use?

## Wayne Grudem handling

Wayne Grudem's *Systematic Theology* is valuable as a conservative evangelical anchor, but the full text should not be ingested into this public repository without a reviewed lawful basis.

Safe handling:

- source profile;
- doctrine order and table-of-contents level map;
- chapter/page references where available;
- original summaries;
- short review quotations only when appropriate;
- no full-text chunks in public repo.

## Bible ingestion order

Do not begin with full Bible ingestion.

Recommended order:

1. Bible source and licensing registry.
2. Canonical reference format.
3. Biblical connection vocabulary bridge.
4. One-book open-text pilot, such as Romans or John.
5. Original-language alignment pilot.
6. Claim/evidence anchors.
7. RAGGraph/NOS rendering.

## Strategic posture

The source spine should make Logos credible to conservative churches by being orthodox, source-aware, license-aware, transparent about denominational scope, and careful about not making RAG output equal to doctrine.

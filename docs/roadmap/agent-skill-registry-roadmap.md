# Agent / Skill Registry Roadmap

## Phase 1: governance docs

Create policies, templates, taxonomy, and edge vocabulary.

## Phase 2: card registry

Add machine-readable cards after schema route approval.

## Phase 3: validation

Add validators for required metadata, permissions, source boundaries, and drift checks.

## Phase 4: derived discovery

Generate indexes, embeddings, overlap candidates, duplicate candidates, and GraphRAG summaries.

## Phase 5: tool-specific adapters

Generate or map selected cards to Claude Skills, Claude subagents, Codex workflows, OpenAI-style agents, or local model runners.

## Phase 6: runtime orchestration

Only after policy, schema, validators, run ledger, approval gates, and security checks.

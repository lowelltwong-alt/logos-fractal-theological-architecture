# Chat-Based AI Guide

## Appropriate uses

Use chat-based AI for:

- brainstorming;
- research synthesis;
- prompt drafting;
- PR description drafting;
- review checklists;
- source maps;
- staged research drafts;
- patch planning;
- red-team review.

## Inappropriate claims

A chat-based AI must not claim it pushed, merged, validated, or changed the repository unless it actually has live repository tools and performed those actions.

## Required handoff posture

When chat output is meant for the repo, convert it through the chat handoff route or staged research route.

Chat output should not silently become:

- doctrine;
- concept truth;
- claim objects;
- graph objects;
- source registry truth;
- ingestion output;
- runtime agent behavior.

## Internet use

Chat-based AI should browse only when current facts, official tool documentation, source status, or license verification are required.

## Source boundaries

Do not paste or commit protected source text, private licensed corpora, secrets, credentials, privileged material, or sensitive personal data.
